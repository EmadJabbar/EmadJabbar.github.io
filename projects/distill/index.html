Multi-label classification of the MSDialog dataset with DistilBERT

Overview

This project was a part of my B.Sc thesis. In this project, I used the DistilBert for Multi-label classification of the MSDialogue dataset.
The MSDialog dataset is a labeled dialog dataset of question answering (QA) interactions between information seekers and answer providers from an online forum on Microsoft products (Microsoft Community). The dataset contains more than 2,000 multi-turn information-seeking conversations with 10,000 utterances that are annotated with user intent on the utterance level (Each utterance can be assigned multiple labels because an utterance can be associated multiple intents). MSDialog has several versions, including the complete set (MSDialog-Complete) and a labeled subset (MSDialog-Intent). We also preprocessed the data to produce MSDialog-ResponseRank for conversation response ranking. (visit https://ciir.cs.umass.edu/downloads/msdialog/ for more information)
